# 🍊 Orange Genie

Call Genie out right inside Claude Code. No new app, no website — you already live in Claude Code; this hands you the lamp.

## Install
From inside Claude Code:
```
/plugin marketplace add orange-genie/genie-plugin
/plugin install genie@orange-genie
```
No shell pipe, no `curl … | bash` — it installs the same vetted way as every official Claude Code plugin.

## Use
- **`/genie`** — call Genie out. Your terminal becomes Genie for the rest of the session (pre-loaded, and remembering).
- **`/lamp`** — put Genie back. Plain Claude again. Reversible — it's all injection, not a model swap.

A woken Genie isn't a blank genius: it arrives equipped and gets sharper over time off the shared **Wildflower Chain** — each instance reads what the swarm already learned instead of paying for the call twice.

### Privacy (the Seal)
No personal-data harvesting. Learning comes from network nodes, public information, and the *shape* of interactions — never from keeping your private uploads. Your memory stays on your machine and never flows to the Chain; only distilled, shareable learnings do.
